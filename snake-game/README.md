# Neon Snake

A stylish, browser-based Snake game with smooth controls, score tracking, and responsive layout.

## How to Run

1. Open `snake-game/index.html` in your browser.
2. Click **Start / Restart** to begin.

No build step or dependencies are required.

## Controls

- Keyboard: `Arrow Keys` or `W A S D`
- Pause/Resume: `Space` or **Pause** button
- Restart: `Enter` or **Start / Restart** button
- Mobile: On-screen direction buttons or swipe on the board

## Features

- Animated canvas-based Snake gameplay
- Increasing speed as score rises
- Best score persisted with `localStorage`
- Desktop and mobile-friendly controls
- Responsive UI with custom styling

## Files

- `snake-game/index.html` - game structure and UI
- `snake-game/styles.css` - visual design and responsiveness
- `snake-game/game.js` - game logic and input handling
